# 14 | Practice Exercises: Web Basics

Test your knowledge by completing these challenges. Try to do them without looking back at the guides first!

---

## Challenge 1: The News Article (HTML)
Create a page named `article.html` that mimics a news site.
- **Criteria:**
  - A large `<h1>` for the headline.
  - A `<p>` with the author's name in `<i>` (Italics).
  - Use `<hr>` after the intro.
  - Use an `<img>` with a caption using `<figcaption>` (Search this tag online!).
  - Use `<ul>` for "Key Points" of the article.

---

## Challenge 2: The Pricing Table (Tables & CSS)
Create a pricing table for a subscription service.
- **Criteria:**
  - 3 Columns: **Plan**, **Price**, **Features**.
  - Use `<thead>` for the headers.
  - Make the "Pro" plan row have a light yellow background color in CSS to highlight it.
  - Center all text in the table using `text-align: center;`.

---

## Challenge 3: Registration Form (Forms)
Create a sign-up form for an event.
- **Criteria:**
  - Input for **Full Name** (Text).
  - Input for **Email** (Email).
  - Radio buttons for **Ticket Type** (VIP, Regular, Student).
  - Select dropdown for **Shirt Size** (S, M, L, XL).
  - Textarea for **Food Allergies**.
  - A **Submit** button.

---

## Challenge 4: The Navigation Bar (Flexbox)
Create a navigation bar that sticks to the top of the screen.
- **Criteria:**
  - The logo should be on the left.
  - Links (Home, About, Services) should be on the right.
  - Use `justify-content: space-between;`.
  - Add a hover effect that changes the background color of the links smoothly.

---

## Challenge 5: Responsive Cards (Grid & Media Queries)
Create 3 "Profile Cards" for team members.
- **Criteria:**
  - Desktop: 3 columns (side by side).
  - Mobile (< 600px): 1 column (stacked).
  - Each card should have a `border`, `padding`, and `border-radius`.
