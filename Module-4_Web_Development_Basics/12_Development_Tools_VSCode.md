# 12 | Development Tools: Mastering VS Code

To become a professional developer, you need the right tools to write code faster and with fewer errors.

---

## 1. Why VS Code?
Visual Studio Code is the industry standard because it is lightweight, fast, and has thousands of extensions.

### ⌨️ Important Shortcuts
- `Ctrl + S`: Save (Do this constantly!).
- `Ctrl + /`: Comment out code.
- `Ctrl + P`: Find a file quickly.
- `Tab`: Indent your code correctly (Be tidy!).

---

## 2. Essential Extensions for Web Dev
Go to the **Extensions** icon on the left (or press `Ctrl+Shift+X`) and install:

| Extension | Purpose |
| :--- | :--- |
| **Live Server** | Launch a local server and see your changes instantly as you save. |
| **Prettier** | Automatically formats your code to look professional. |
| **Auto Rename Tag** | Renames the closing tag automatically when you change the opening tag. |
| **Color Highlight** | Shows the actual color in your CSS next to the hex code. |

---

## 3. Using Emmet (The Secret Sauce)
Emmet is built-in. It lets you write complex HTML with just a few characters.

- **Instead of typing:** `<div id="box" class="container"></div>`
- **Type:** `div#box.container` and press `Tab`.

- **Instead of typing:** A list with 5 items.
- **Type:** `ul>li*5` and press `Tab`.

---

## 4. Browser Dev Tools (F12)
Your browser is a development tool! Right-click any element and select **"Inspect"**.
- You can change colors and sizes in live-time.
- You can see how your site looks on a Mobile Device (Click the phone/tablet icon).
- **Console:** Shows you errors (vital for JavaScript later).
